# NovaCampus

NovaCampus is a polished, futuristic landing page and demo interface for a metaverse-native education platform. It features an immersive brand experience, animated particle effects, a multi-page navigation flow, student and teacher dashboard mocks, a virtual classroom section, and a course marketplace.

## Live Demo
This project is deployed on GitHub Pages.

> Visit the live site at: `https://<your-github-username>.github.io/<repository-name>/`

## Features

- Responsive single-page layout with a fixed navigation bar
- Animated hero section and particle background
- Student and teacher authentication pages
- Student & teacher dashboard mockups with cards, progress visuals, and analytics
- Virtual classroom UI with chat panel and live session controls
- Course marketplace with filters, trending cards, and institution sections
- Community and gamification pages for achievements and social interaction

## Files

- `index.html` — main page structure and UI content
- `style.css` — visual styling and animations
- `nos.js` — page navigation, interactions, particle generation, and UI state

## Run Locally

1. Clone the repository.
2. Open `index.html` in your browser.

Alternatively, use a simple local server for best results:

```bash
# Python 3
python -m http.server 8000
```

Then open `http://localhost:8000`.

## GitHub Pages Deployment

To deploy this repository to GitHub Pages:

1. Push the code to your GitHub repository.
2. In the repository settings, enable GitHub Pages.
3. Select the branch (`main` or `master`) and `/ (root)` as the publishing source.
4. Save changes and wait for the site to publish.

## Notes

- This project is a frontend prototype and does not include backend authentication or live data.
